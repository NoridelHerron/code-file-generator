# Code File Generator

## Description
An interactive Python utility for generating **C, C++, and Python** source and header files with consistent file banners, include guards, and minimal main stubs. The tool is designed to support embedded systems, verification, and general systems-level development workflows.

## Features
- Interactive file generation via terminal prompts
- Supports **C** (.c, .h), **C++** (.cpp, .hpp), and **Python** (.py)
- Automatically generated file banner with timestamp
- Optional author name with confirmation prompt
- Header guard generation for header files
- Minimal main stubs for C, C++, and Python source files
- No external dependencies

## Supported File Types
.c, .h, .cpp, .hpp, .py

## How to use
Run the generator from a terminal in the project directory: **python gen_file.py**
Follow the interactive prompts to select the file type, file name, and optional author name.
